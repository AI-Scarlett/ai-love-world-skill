## 🛒 购买步骤

### 第 1 步：访问阿里云

```
https://www.aliyun.com
```

登录账号（没有就注册一个）

---

### 第 2 步：选择云服务器 ECS

1. 点击顶部菜单 **"产品"** → **"云服务器 ECS"**
2. 点击 **"立即创建"** 或 **"免费试用"**（新用户）

---

### 第 3 步：选择计费方式

**推荐：按量付费**（灵活，可随时升级）

| 计费方式 | 适合场景 | 价格 |
|---------|---------|------|
| 按量付费 | 测试、初期 | 按小时计费 |
| 包年包月 | 长期稳定使用 | 更便宜 |
| 抢占式实例 | 临时测试 | 最便宜但不稳定 |

**新用户优惠：**
- 首年 ¥60-100（1 核 2G）
- 限时特价，很划算！

---

### 第 4 步：选择地域

**推荐：华东 1（杭州）或 华东 2（上海）**

| 地域 | 优点 |
|------|------|
| 华东 1（杭州） | 离丝佳丽近，延迟低 |
| 华东 2（上海） | 网络质量好 |
| 华北 1（青岛） | 北方用户访问快 |
| 华南 1（深圳） | 南方用户访问快 |

**选择原则：** 离你的用户群体近

---

### 第 5 步：选择实例规格

**展开"进阶配置"，选择：**

```
实例规格族：突发性能实例 t6
规格：ecs.t6-c1m1.large
配置：1 核 CPU | 2GB 内存
```

**为什么选 t6？**
- ✅ 便宜
- ✅ 适合 Web 应用
- ✅ 有 CPU 积分机制
- ✅ 初期完全够用

**后续升级路径：**
```
t6-c1m1.large (1 核 2G) → 
t6-c1m2.large (1 核 4G) → 
c6.large (2 核 4G) → 
c6.xlarge (4 核 8G)
```

---

### 第 6 步：选择镜像（重要！）

**强烈推荐：Ubuntu 22.04 LTS 64 位**

**选择路径：**
```
镜像市场 → 公共镜像 → Ubuntu → Ubuntu 22.04 LTS 64 位
```

**为什么选 Ubuntu 22.04？**
- ✅ 长期支持版（5 年安全更新）
- ✅ Python 3.10 预装
- ✅ 文档丰富
- ✅ 丝佳丽的代码基于 Ubuntu 开发
- ✅ 社区活跃

**不要选：**
- ❌ CentOS 7/8 - 已停止维护
- ❌ Windows Server - 太贵，不适合 Python
- ❌ 太老的版本 - 安全更新少

---

### 第 7 步：存储配置

**系统盘：**
- 类型：ESSD 云盘
- 容量：40GB（默认）
- 性能：PL0 即可

**数据盘：** 初期不需要，后续可加

---

### 第 8 步：网络配置

**带宽：**
- 计费方式：按使用流量（推荐）或 按固定带宽
- 带宽值：1-3Mbps（初期够用）

**按流量计费：**
- 优点：用多少付多少，便宜
- 价格：约 ¥0.8/GB
- 适合：流量不稳定的应用

**按带宽计费：**
- 优点：费用固定
- 价格：1Mbps 约 ¥23/月
- 适合：流量稳定的应用

**推荐：按使用流量**（初期更省钱）

---

### 第 9 步：安全组配置

**开放端口：**
- ✅ 22 (SSH) - 远程登录
- ✅ 80 (HTTP) - Web 访问
- ✅ 443 (HTTPS) - 加密访问
- ✅ 8000 (API) - 主 API 服务
- ✅ 8002 (Auth) - 认证服务

**创建安全组规则：**
```
方向：入方向
协议：TCP
端口：22, 80, 443, 8000, 8002
授权对象：0.0.0.0/0
```

---

### 第 10 步：系统配置

**登录凭证：**
- 选择：**自定义密码**
- 用户名：root
- 密码：设置一个强密码（记录下来！）

**或者：**
- 选择：SSH 密钥对（更安全）
- 需要提前创建密钥对

---

### 第 11 步：确认订单

**检查清单：**
- [ ] 实例规格：1 核 2GB
- [ ] 系统：Ubuntu 22.04 LTS
- [ ] 磁盘：40GB ESSD
- [ ] 带宽：1-3Mbps
- [ ] 安全组：已开放必要端口
- [ ] 密码：已设置并记录

**确认无误后：**
- 点击 **"立即开通"**
- 完成支付

```bash
chmod +x ubuntu_setup.sh
sudo ./ubuntu_setup.sh
```

**4. 上传项目代码**

```bash
supervisorctl restart all
```
